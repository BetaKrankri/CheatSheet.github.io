Referencia de Elementos HTML

Esta página lista todos los elementos HTML. Están agrupados por funciones para ayudarte a encontrar lo que tienes en mente con facilidad. Aunque esta guía está escrita para aquellos que son nuevos escribiendo código, se pretende que sea una referencia útil para cualquiera.

Raíz Principal

Elemento Descripcion

<html>

 Este elemento HTML representa la raíz de un documento HTML. El resto de elementos descienden de este elemento

Metadatos del documento

Los metadatos del documento contienen información sobre la página. Esto incluye información sobre estilos, scripts y datos que ayudan al software ( search engines ,browsers , etc.) a usar y generar la página. Los metadatos de estilos y scripts pueden estar definidos en la página o estar enlazados a otro fichero que contiene la informacion.

Elemento Descripcion

<base>

 El elemento <base> especifica la direccion URL base que se utilizara para todas las direcciones URL relativas contenidas dentro de un documento. Sólo puede haber un elemento <base> en un docuemnto.

<head>

 El elemento <head> provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.

<link>

 el elemento <link> especifica la relación entre el documento actual y un recurso externo. Este elemento es usado comunmente para ligar al CSS, pero tambien se usa para establecer el icono del sitio (favicon).

<meta>

 El elemento <meta> representa los metadatos que no pueden ser representados por otro elemento HTML "meta-relacionado", como base, link, script, style, o title.

<style>

 El elemento <style> contiene la informacion del estilo de un documento. Contiene CSS, que se aplica a los elementos del documento especificado dentro del elemento <style>.

<title>

 El elemento <title> define el titulo del documento que se muestra en la barra de titulo del Navegador. Solo contiene texto, las etiquetas dentro de este elemento serán ignorados.

Raíz de seccionamiento

Elemento Descripcion

<body>

 El elemento <body> representa el contenido de un docuemnto HTML. Solo puede haber un eleneto <body>.

Seccionamiento de Contenido

Los elementos de seccionamiento de contenido permiten organizar el contenido del documento en partes lógicas. Usar los elementos de seccion ayuda a crear un esquema general para el contenido de la página.

Elemento Descripcion

<address>

 El elemento <address> indica que el HTML contiene informacion de contacto.

<article>

 El elemento <article> representa una composicion autónoma en un documento, página, aplicación o sitio, que está destinada a ser distribuible o reutilizable de forma independiente.

<aside>

 El elemento <aside> representa una porcion del documento cuyo contenido es indirectamente relacionado con el contenido principal del documento.

<footer>

 El elemento <footer> representa un pie de página. Normalmente contiene informacion sobre el autor de la seccion, copyright o links hacia documentos relacionados.
<h1>, <h2>,
<h3>, <h4>, <h5>, <h6> 		Los Elementos <h1> a <h6> representan seis niveles de encabezados de seccion. <h1> es el encabezado de mayor nivel, mientras que <h6> es el de menor.
